## What is CTF?


## CTF

* CTF stands for Capture The Flag
* Security competitions where you solve hacking challenges
* Solo or as a team
* Goal is to capture flags


**FLAG{THIS_IS_A_FLAG}**


### Example

* Web server listening on port 8080
* Navigate to the webserver
* Exploit a vulnerability
* Retrieve a flag
